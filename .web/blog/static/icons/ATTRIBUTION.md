## Compose Icon Attribution

The compose icon cycler uses:
- A base set of direct quill-focused icons from Iconify collections.
- 55 real quill/writing SVGs sourced from Openclipart (stored as `compose-oc-01.svg` .. `compose-oc-55.svg`).
- 10 quill/ink SVGs sourced from Wikimedia Commons (stored as `compose-cm-01.svg` .. `compose-cm-10.svg`).
