## Compose Icon Attribution

The compose icon cycler now includes only:
- `streamline-pixel:content-files-quill-ink` (Iconify collection)
- `hugeicons:quill-write-02` (Iconify collection)
- 20 in-repo custom SVG files (`custom:quill-icon-01` .. `custom:quill-icon-20`) served from `/static/icons/quill-icon-01.svg` … `/static/icons/quill-icon-20.svg`. Each asset is a simplified download from Iconify collections (Streamline Freehand/Pixel, Hugeicons, TDesign, MingCute, Remix, Phosphor, Game Icons) that keeps a paper + nib/pen motif at navbar scale.

Usage note: navbar rendering displays compose icons in monochrome black.
