## Compose Icon Attribution

The compose icon cycler now includes only:
- `streamline-pixel:content-files-quill-ink` (Iconify collection)
- `hugeicons:quill-write-02` (Iconify collection)
- 20 in-repo custom SVG variants (`custom:quill-page-01` .. `custom:quill-page-20`) generated in `nav-auth.js`; each draws a simple paper shape with a quill nib poking out the upper-right corner so their silhouette matches the requested vibe.

Usage note: navbar rendering displays compose icons in monochrome black.
