ℹ️ About This App

This chatroom app demonstrates wizardry's desktop app framework:

  • Uses the same CGI backend as the web demo
  • Server-Sent Events (SSE) for real-time updates
  • Tor integration for anonymous access
  • Commands hardcoded in GUI for security
  • Compatible with MUD say command format
  • Fully interoperable: web users and MUD players share the same rooms!
