# Concurrency Race Condition Fix
This directory contains scripts to demonstrate a race condition in JavaScript, fix it using locks, and verify that the fix works through stress testing.
## Files
- **.race-condition-fix.js**: Contains the fixed code with lock mechanism to avoid race conditions.
- **.test-stress.js**: Runs multiple iterations of stress tests to reproduce and test the race condition.
- **.verify-fix.js**: Verifies that the fix works by running stress tests and checking the final state of sharedResource.
